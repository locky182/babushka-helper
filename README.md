# Дневник Давления ❤️

Мобильное приложение на Flutter для простого контроля артериального давления. Специально разработано с учетом потребностей пожилых людей: крупные шрифты, контрастные цвета и понятный интерфейс.

🚀[СКАЧАТЬ ПРИЛОЖЕНИЕ (APK)](https://github.com/locky182/babushka-helper/releases/download/v3.0/app-armeabi-v7a-release.apk)

## ✨ Основные функции
- ✅ Добавление замеров (систолическое, диастолическое давление и пульс).
- ✅ Группировка истории по дням.
- ✅ Автоматический расчет среднего значения за день.
- ✅ Цветовая индикация состояния ("Светофор").
- ✅ Удаление ошибочных записей свайпом.
- ⏳ В разработке: Графики динамики за неделю.
- ✅ Возможность добавления препаратов и дозировки
- ✅ Автокомпликт препаратов ранее введенных
- ✅ Динамика в виде графики
- ✅ Возможность поделиться с врачом таблицей данных и препаратов.

🚀 Release v3.0: Семейный Доктор & Индивидуальный ПодходЭтот релиз превращает простой дневник в полноценную систему мониторинга здоровья для всей семьи.
Мы прислушались к отзывам наших главных тестеров (наших родных!) и сделали приложение еще человечнее.👥 Мультипрофильность и Личные ЦелиКарточки пациентов: Теперь можно создавать профили для каждого члена семьи (папа, мама, бабушка) с указанием имени, пола и возраста.
Целевое давление: У каждого профиля теперь есть поле «Моя норма». Это позволяет учитывать индивидуальные особенности организма и рекомендации личного врача.
Индикация в списках: В меню выбора профиля сразу видно возраст и целевые показатели.🩺 Умный мониторинг пульсаЦветовая дифференциация: Пульс теперь не просто цифра. Мы внедрили медицинские стандарты:🟦 Синий: < 50 (брадикардия)🟩 Зеленый: 50–90 (норма)🟧 Оранжевый: 91–110 (повышенный)🟥 Красный: > 110 (тахикардия)График пульсации: На экране статистики добавлен отдельный график пульса под графиком давления. Они синхронизированы по времени для удобства анализа.
Новые иконки: В карточках записей появилась иконка кардиограммы (monitor_heart), чтобы не путать показатели сердца с общим статусом давления.📄 Профессиональные PDF-отчетыПерсонализация: Теперь в заголовке отчета указано имя и возраст пациента.Целевые показатели: Врач сразу увидит в PDF, на какое давление ориентируется пользователь.Чистота данных: Отчеты стали более структурированными и читаемыми для врачей.

Теперь у каждого поля (систола, диастола, пульс) есть свой отдельный микрофон для максимально точного ввода голоса.Валидация: Добавлена защита от случайного ввода букв или пустых полей.
